
# 金庸

金庸 https://zh.wikipedia.org/wiki/%E9%87%91%E5%BA%B8
- > 其人
  * > 接受访谈时，金庸认为自己像张无忌（《倚天屠龙记》），妻子像夏青青（《碧血剑》）。
  * > 金庸在《鹿鼎记》后记中，表示自己最喜欢的作品是《倚天屠龙记》、《笑傲江湖》、《神雕侠侣》和《射雕英雄传》(在新修版《鹿鼎记》中的后记加入了《天龙八部》)。
  * > 金庸在《倚天屠龙记》后记中，表示自己最爱的女性人物是小昭（《倚天屠龙记》）。
