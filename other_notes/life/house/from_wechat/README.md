# from 通货朋仗

## 2021.09

两年后的北京房价…… https://mp.weixin.qq.com/s/gJSlY8IfEye_x-Pa11bFjw
- > 1，今年北京的前两批住宅供地，供应了73块，其中第一批是30块地，第二批是43块地。后边还有一批供地，如果数量也足够多的话，那么今年的供地应该能超过100块。这个量挺大的。
- > 2，第一批供地和第二批供地的玩法上发生了比较大的变化。不过总体思路上还是：地价有上限，房价有上限，在这个基础上尽量提高住宅质量。
- > 我说下这次的变化吧。
  * > 第一，这次有几个项目增加了摇号确定开发商的环节，比如说，三家开发商拍一块地，都拍到了上限价格，这个时候就直接摇号，谁运气好谁拿地。 <br> 摇号应该属于试点玩法，政府也想看下效果咋样。不过既然摇号确定，那摇号的门槛肯定还是比较高的，也就是说，最终能拿到这种项目的开发商，实力肯定没问题，最后的产品交付应该也比较满意。
  * > 第二，不再竞公共租赁住房面积。第一批供地的时候，40%的楼盘都在竞公租房面积。这就导致好多项目，既有商品房，又有公租房。其实项目这么搞，对于将来的小区品质各方面，都不好。而且，拿公租房面积当成开发商竞地的一种筹码，也有点问题。 <br> 这次直接给几个地块规定了租赁住房面积，不用竞争。
  * > 第三，有7090政策的项目特别少，开发商可以放飞自我了。
- > 4，北京的供地位置，一言难尽。***好的地块不往外拿，都扔一些边边角角。而且边边角角的位置，价格也不低***。这就导致了，**绝大多数的楼盘，未来卖房的时候，房子不会太好卖**。
