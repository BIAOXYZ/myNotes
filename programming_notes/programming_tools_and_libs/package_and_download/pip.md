# `requirement.txt`

初学Python-使用requirement.txt https://www.jianshu.com/p/d3e272243cd4
- > 第一步：自动生成requirement.txt，命令：
  >> `pip freeze > requirements.txt`
- > 第三步：安装requirement.txt，在新环境，执行命令即可一键安装完所需要的第三方库。命令：
  >> `pip install -r requirements.txt`
