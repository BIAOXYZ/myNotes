
# DDD (Data Display Debugger)

```sh
yum -y groupinstall "X Window System"
mkdir dddpackage
cd dddpackage/
wget https://rpmfind.net/linux/epel/7/x86_64/Packages/d/ddd-3.3.12-20.el7.x86_64.rpm
yum install -y ddd-3.3.12-20.el7.x86_64.rpm
```
