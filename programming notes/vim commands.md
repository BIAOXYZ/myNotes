# 1.general vim commands part:
----------------------------------------------------------------------------------------------------

## vim设置和取消显示行号
:set nu  
:set nonu

```
最好的办法是直接修改/etc/vimrc文件，在末尾添加
set nu
```

## vim显示当前正在编辑的文件名和路径
:f 或 CTRL+G
```
:args 只显示文件名不显示路径
```

## vim移动命令
### vim跳转到文件头和文件尾
:1 或 gg
:$ 或 G

## 查找和替换

```
:1,$s/old/new/g 
将文件中所有的『old』改成『new』。
```
