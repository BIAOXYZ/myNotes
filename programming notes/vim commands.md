## vim设置和取消显示行号
:set nu  
:set nonu

```
最好的办法是直接修改/etc/vimrc文件，在末尾添加
set nu
```

## vim显示当前正在编辑的文件名和路径
:f 或 CTRL+G


## vim移动命令
### vim跳转到文件头和文件尾
:1 或 gg
:$ 或 G
